## ✨ Features

- 🔐 **Role-Based Access** — Student, Club Owner, Admin
- 📅 **Google Calendar Sync** — Auto-add events on RSVP
- 🤖 **Gemini AI Assistant** — Natural language event queries
- 🔔 **Push Notifications** — Local reminders 10 min before events
- 📊 **Analytics Dashboard** — Branch-wise participation heatmaps
- 🌙 **Dark Mode** — System-aware with persist preference
- 💳 **Payments & Wallet** — Paid event support
- 📱 **QR Check-in** — Scanner for attendance

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/roshankumar0036singh/Uni-Event.git
cd Uni-Event/app

# Install dependencies
npm install

# Start the dev server
npx expo start
```
