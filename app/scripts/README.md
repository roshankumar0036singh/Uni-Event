# Scripts

This folder contains utility and diagnostic scripts used during development and maintenance of the Uni-Event app.

## Scripts Overview

| Script | Purpose |
|--------|---------|
| `copy_externals.js` | Copies external dependencies |
| `diagnose_externals.js` | Diagnoses issues with external modules |
| `diagnose_tap.js` | Diagnoses tap related issues |
| `dump_externals.js` | Dumps external module information |
| `grep_externals.js` | Searches through external dependencies |
| `inspect_externals.js` | Inspects external module details |
| `patch_expo.js` | Patches Expo configuration |
| `patch_expo_v2.js` | Updated version of Expo patch script |
| `patch_expo_final.js` | Final version of Expo patch script |
| `patch_is_generator.js` | Generates patch configurations |
| `read_critical.js` | Reads critical dependency information |
| `read_dependency.js` | Reads dependency details |
| `read_externals.js` | Reads external module data |
| `read_externals_v2.js` | Updated version of read externals script |
| `regenerate_externals.js` | Regenerates external module configurations |
| `verify_status.js` | Verifies app status and configuration |
| `seed_early_bird_even....js` | Seeds early bird event data |

## Usage

These scripts are development tools only. Do not run them in production.
Run individual scripts using:
```
node scripts/<script-name>.js
```
